To embed a subscribe form from beehiiv on your website, you can follow these steps:
  • WordPress: Use a plugin like WPForms or add a Custom HTML block in the editor to paste your beehiiv subscribe form embed code.
  • Shopify: Go to Online Store > Pages, select the page, and paste the embed code in the HTML editor.
  • Squarespace: Use the Embed block in the editor or use Code Injection for site-wide placement.
  • Wix: Use the Custom Embeds option to add your beehiiv form.
  • Other platforms: Insert the embed code into your site's HTML or use an HTML/code block.

how do i change the name for my url
To change the name of your beehiiv URL, follow these steps:
  1. Go to your Dashboard and select Settings.
  2. Click on Domains from the left panel.
  3. Under beehiiv Subdomains, click Edit next to your web URL.
  4. Enter your new URL name, using hyphens for spaces or removing spaces entirely.
  5. Click Confirm to save your changes.
Keep in mind that changing your web URL will also update your email username and affect links to existing posts and backlinks. For more details, please visit this link: https://beehiivhelp.zendesk.com/hc/en-us/articles/18203673055767-Changing-the-name-email-username-or-web-URL-of-your-publications
